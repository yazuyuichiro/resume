## 副業プロジェクト実績

### 医療系データを用いた予測モデル構築
- 医療系の画像特徴量（説明変数約80個）と2値の目的変数を用いた教師あり学習によるクラス分類モデルの構築・評価
- 特徴量選択、機械学習モデル構築、交差検証（K-Fold・LOOCV）、古典的統計解析との比較
- **使用技術：** Python、scikit-learn、統計解析ツール

### 過去データを用いた条件別集計・回収率分析ツール開発
- 過去5〜10年分の時系列記録データから、指定条件に合致するケースを抽出し回収率を算出するツールの開発
- データ収集・整形、条件フィルタリングロジックの実装、集計・可視化
- **使用技術：** Python

### 動画テロップ自動挿入ツールの開発
- 任意の動画ファイルに対し、テキストや画像（矢印等）を指定位置に挿入するスクリプトの実装
- moviepyを用いたテロップ・画像合成処理。文字・位置を容易に変更できる設計
- **使用技術：** Python、moviepy

### 画像・動画生成AIの検証・調査支援
- 最新の画像生成・動画生成AIツールの動向調査、検証、活用方法のリサーチ支援
- 各種生成AIツールの検証、ユースケース整理、調査レポート作成
