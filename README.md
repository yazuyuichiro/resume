## プロジェクト実績（抜粋）

### 【副業】特定企業向けAI導入コンサルティング・伴走型ツール開発
- 週次定例MTGを実施し、現場の課題ヒアリング、AI活用のアドバイス、社内リテラシー向上のための教育を牽引
- ヒアリング内容を基に、以下の各種ツールの要件定義・アーキテクチャ設計・プロトタイプ実装・改善をアジャイルに実行
  - 生成AIを活用したコンテンツ生成Webアプリのプロトタイプ開発
  - ブラウザ自動化およびスクレイピングによる営業・情報収集プロセスの自動化システム
  - 社内リテラシー向上を目的としたAI・IT関連教材の作成
- **使用技術：** Python, Streamlit, OpenAI API, Anthropic API, Gemini API, Claude Code, Gemini CLI

### 【本業】生成AIを活用した社内・顧客向けシステム開発（PM・実装）
- LLMを活用したテキストデータの自動評価・分析システムの開発
- 独自アルゴリズムを用いたマッチングシステムおよび自動データ生成ツールの開発
- 業務効率化に向けた画像処理AIパイプラインの構築
- **使用技術：** Python, 各種LLM API, データ分析系ライブラリ

### 【副業】ヘルスケア領域の実データを用いた予測モデル構築
- 特定業界の画像特徴量（多数の説明変数）と2値の目的変数を用いた教師あり学習によるクラス分類モデルの構築・評価
- 特徴量選択、機械学習モデル構築、交差検証（K-Fold・LOOCV）、古典的統計解析との比較
- **使用技術：** Python, scikit-learn, 統計解析ツール

### 【個人開発】AIエージェント活用・データ分析
- AIエージェントを用いた自動情報収集および動画・コンテンツ作成ツールの構築
- スポーツデータの収集・統計分析および可視化
- **使用技術：** Python, Claude Code, Gemini CLI, Cursor
